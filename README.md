# Smart AI Image Understanding System

An AI-powered web application that generates natural language descriptions from uploaded images using BLIP.

## Features
- Upload any image
- AI-generated image captions
- Gradio web UI
- Google Colab ready
- Real-time output

## Tech Stack
- Python
- BLIP
- Transformers
- PyTorch
- Gradio

## Run Locally
```bash
pip install -r requirements.txt
python app.py
